# stock-prediction
## Python based application for comparing stock prediction between ML model (ARIMA) and DL model (LSTM)
### Streamlit – GUI
### Keras + Tensorflow – LSTM, RNN
### Statsmodels – ARIMA
### Scikit Learn – Error Calculation
### Matplotlib – Visualisaiton
